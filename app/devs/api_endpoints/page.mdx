---
Ez itt az API végpontok tartalomjegyzéke.

Minden végpontot egy adott controller alatt van.

> [!NOTE]
>
> FONTOS: minden végpont az alábbi cím - port alatt van: localhost:5110/

| Végpont neve | Rövid leírás |
| ------------ | ------------ |
| /api/User (UserController) | Ez a végpont felel a felhasználók kezeléséért |
| /api/Post (PostController) | Ez a végpont felelős a posztok kezeléséért |
| /api/Group (GroupController) | Ezen a végponton van az összes csoportokhoz tartozó műveletek |
| /api/Auth (AuthController) | Felelős a felhasználók autentikációjáért |
