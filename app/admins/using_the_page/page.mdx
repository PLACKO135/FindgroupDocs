## Kezdőoldal / Bejelentkezés

A kezdőoldal egyből a bejelentkezési oldal az adminoknak.

## Admin felület

Bejelentkezés után egyből az admin felület fogad. Itt az admin tudja menedzselni a:

- Felhasználókat
- Csapatokat
- Csapat hírdetéseket

Ezenkívűl tudja azokat exportálni .csv formátumban.
