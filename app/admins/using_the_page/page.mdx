# Oldal használata

## Kezdőoldal / Bejelentkezés
>[!NOTE]
>
>Az oldalon látható fekete négyzetek felhasználókat takarnak

A kezdőoldal egyből a bejelentkezési oldal az adminoknak.
<br/>

<img src ="/AdminloginDoc.png" alt="Bejelentkezés" title="Bejelentkezési oldal"/>

## Admin felület

Bejelentkezés után egyből az admin felület fogad. Itt az admin tudja menedzselni a:

- Felhasználókat
- Csapat hírdetéseket
- Csapatokat


Itt tudja az adminisztrátor kiválasztani hogy mit szeretne tenni az adott felhasználóval, hírdetéssel vagy csapattal;

### Szerkesztés / Törlés

Kettő lehetőség közül választhat az adminisztrátor vagy törölheti a felhasználót, hírdetést, csapatot vagy szerkesztheti annak bizonyos adatait.
